# Fake Store Product Listing

A responsive product listing application built using **React.js**, **Next.js**, and **Bootstrap 5**.

## Features

- Server Side Rendering (SSR) using getServerSideProps
- Product Listing
- Product Details Page
- Search Products
- Client-side Pagination
- Loading Spinner
- Responsive Bootstrap Layout

## Tech Stack

- React.js
- Next.js
- Bootstrap 5
- Fake Store API

## API

https://fakestoreapi.com/products

## Installation

```bash
git clone <repository-url>

cd product-listing

npm install

npm run dev
```

Visit

```
http://localhost:3000
```
